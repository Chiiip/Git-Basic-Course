cheat sheet
<Appear>
<h4>
git clone nomedorepositorio - clona um repositório remoto já existente
</h4>

<h4>
git add nomedoarquivo - adiciona um arquivo à staging area
</h4>
<h4>
git restore nomedoarquivo --staged - remove um arquivo da staging area
</h4>
<h4>
git commit -m "Mensagem do Commit" - cria um commit com os arquivos presentes na staging area
</h4>
<h4>
git push - envia os commits do repositório local para o repositório remoto
</h4>
</Appear>
